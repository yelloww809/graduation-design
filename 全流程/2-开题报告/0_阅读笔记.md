# <2> [2020] A Downscaled Faster-RCNN Framework for Signal Detection and Time-Frequency Localization in Wideband RF Systems

## 文献综述

### 0. 背景

随着物联网（IoT）的出现，我们目前正目睹周围无线设备数量的急剧增加。在未来的无线系统中，如果物联网设备和无人驾驶飞行器（UAV）与手机共存，那么从安全和频谱管理的角度出发，就必须对这些设备加以区分。能够检测和分类异构无线信号并确定其时间频段的科技可以转化为无线产品。例如，如果我们检测到窄带发射器的存在，并在时间与频率上定位无线传输，我们就可以开发商业无线安全产品，以发送关于意外无线设备存在的紧急警报，并或使用窄带信号干扰器来阻断该设备的信号。时间频段的定位还使我们能够获取有关哪些时间频段资源未得到充分利用且受到最小干扰的信息。然后，我们可以构建智能频谱分配产品，以在单位面积内为更多的设备提供服务。例如，如果我们对某一设备的跳频模式进行解码，就可以利用时间延迟技术，将同一组时频资源重新分配给另一台具有相同驻留时间、带宽和跳频周期的设备。这种智能频谱分配产品在免许可证的ISM频段中具有巨大的商业价值，尤其是在我们正经历无线设备密度显著增加的物联网（IoT）时代。为了实现上述应用，我们需要开发一种算法，它能够同时完成两项主要任务：（i）检测宽带无线电频率（RF）频谱中无线信号的存在，以及（ii）确定其时频范围。在这方面，当RFI 由无意义信号引起时，我们提出了一种深度学习解决方案。

# <3> [2022] A New Deep Learning Framework for HF Signal Detection in Wideband Spectrogram

## 文献综述

### 0. 背景

高频（HF）通信在军事、航空、导航等领域发挥着重要作用。HF 频带的带宽约为 27 MHz，与 3 kHz 或更低的 HF 信号带宽相比，其带宽相当宽广。如今，HF 频段包含了大量的信号，再加上多径延迟和衰落的影响，HF 环境变得愈发混乱。本文的目的在于在宽带范围内检测多个感兴趣的 HF 信号，并输出时间-频率定位信息和信号类别。

### 1. 信号检测 领域

传统的信号检测方法是在窄带中进行的，用于检测信号的存在，例如最优最大似然法[1]、次优线性零强迫（ZF）法[2]、最小均方误差（MMSE）法[3]以及基于深度学习的算法[4]、[5]。



随着硬件计算能力的快速发展，覆盖整个频段的信号检测成为可能。分频 + 窄带检测是一种解决整个频段检测的方案，它将整个频段划分为具有固定频率步长的窄带通道，然后实施窄带检测[6]。然而，对数万个通道进行通道化处理在计算上成本极高，而且诸如带宽和频率步长之类的通道化参数对于具有不同带宽和频率间隔的信号来说很难选择。



据我们所知，大多数宽带检测方法都是基于通过短时傅里叶变换（STFT）计算出的频谱图，这种频谱图能够清晰地呈现信号的时间-频率分布。由于频谱图是一种特殊的图像类型，因此在频谱图中进行信号检测的任务与图像中的目标检测非常相似。现有的大多数工作确实遵循了目标检测的理念：在[7]中提出了能量检测与卷积神经网络（CNN）分类器的组合，在[8]至[10]中则采用了诸如 Faster-RCNN、SSD、YOLOv3 等一些经典的基于深度学习的目标检测器，这些方法取得了最先进的性能。对于经典的检测器，从通用的目标检测框架到信号检测框架的修改主要在于通过统计或聚类方法调整先验锚点的大小以适应信号。然而，频谱图中信号的纵横比变化极大，即使是精心设计的先验锚点也难以匹配所有信号。此外，大多数传统检测器是通过内部某个点来定位物体的，而其在卷积神经网络中的感受野无法涵盖具有很长时间长度的信号。上述缺陷导致了传统检测器性能的下降。因此，本文旨在设计一个专门针对基于频谱图的信号检测任务的更合理的框架。

# <4> [2024] Spectrum Sensing via Residual Dilated Network and Horizontal Shift Attention for Cognitive IoT

## 文献综述

### 0. 背景

（Abstract）

随着物联网（IoT）部署的持续增长，各种无线设备和通信技术在工业环境中共存，导致频谱变得拥挤且动态变化。高效的频谱感知对于减少干扰、提高通信可靠性以及确保多种无线技术的无缝共存至关重要。然而，极其密集且多变的信号给复杂多变的信号环境中的信号精确检测、估计和识别带来了挑战。

（Introduction）

由于物联网（IoT）和无线传感器网络的迅速发展，频谱已成为认知系统中的稀缺资源。频谱感知在各种无线应用中发挥着至关重要的作用，包括工业[1]、医疗保健[2]、交通运输[3]等等[4]。例如，它可以用于监测工业设备并检测机器发出的宽带信号中的异常振动[5]、噪音和电磁干扰。这使得能够进行预测性维护、故障诊断，并提高设备的可靠性。

### 1. 信号检测 领域

频谱感知涉及对宽带频谱图中感兴趣信号的检测、**定位和分类**。

传统的频谱感知方法包括能量检测（ED）[7]、最大特征值检测（MED）[8]以及协方差绝对值[9]。

早期的研究探索了接收信号的先验信息以进行检测。例如，Shin 等人[10]开发了一种二维时频匹配滤波器来从频谱图中检测信号。Urkowitz [11] 将信号能量建模为非中心卡方分布，并由此推导出最优检测阈值。Chen [12] 通过用正幂运算代替幅度平方运算来提高 ED 的性能。



Hou 等人 [13] 利用复杂的神经网络进行特征提取，并取得了令人满意的成果。然而，该模型仅用于分类任务，无法定位信号。



利用频谱图中大量已标注的“对象”，深度学习能够在检测、估计和识别方面实现更高的准确率。此外，它为无需先验知识的频谱感知问题提供了一种更高效的解决方案，因此更适合于不合作的复杂电磁环境。



例如，Ke 等人[22]将卷积神经网络（CNN）与长短期记忆（LSTM）相结合，以提取频率和空间特征来检测信号。

Huang 等人[23]将信号检测转化为宽带频谱图的语义分割，并设计了一个全卷积网络。

Prasad 等人[24]使用更快的 RCNN 来检测信号，并在短时傅里叶变换（STFT）中分析了超参数。

Li 等人[25]提出了一种无锚点的框架，利用每个频率下的所有时间点特征来预测中心频率和每个信号的形状属性。

### 2. 计算机视觉 目标检测 领域

如今，基于深度学习的宽带信号检测方法已有所发展，例如基于锚点的检测器[14]、[15]、[16]、[17]、[18]以及无锚点的检测器[19]、[20]、[21]。



近年来，视觉转换器（ViTs）[28] 已迅速成为图像分类领域的佼佼者，取代了卷积神经网络（CNNs）。具有层次结构的 ViTs（如 Swin 变换器[29]）的出现，进一步巩固了转换器架构作为适用于各种视觉任务的通用模型的地位。然而，仔细研究 Swin 变换器的设计（其结果与传统 CNNs 相似），刘等人提出了 ConvNext，这是一种完全依赖卷积神经网络的视觉模型，其性能优于 Swin 变换器。受 ConvNext 的启发，我们设计了一种专为宽频信号感知而定制的新型网络，该网络结合了 RDN 和 HSA 技术。此网络的架构在图 6 中有所展示。这种创新设计充分利用了 ConvNext 中所观察到的优势，以提升宽频信号感知的能力，为物联网应用开辟了一条有前景的途径。



由陈等人[41]提出的膨胀卷积方法证明了其在扩大卷积核的感知范围方面是有效的。然而，将其直接应用于信号检测任务却带来了挑战。过高的膨胀率可能会导致特征从特征图上的不相关区域被提取出来。此外，膨胀卷积可能会忽略“小”信号，从而严重影响召回率。



挤压激发（SE）方法[43]是一种开创性的技术，它通过全局池化来计算通道注意力，但忽略了位置信息。卷积块注意力模块（CBAM）[44]试图通过在通道上进行全局池化来引入位置信息。然而，它主要捕捉局部信息，无法理解长距离的依赖关系。坐标注意力（CA）[45]成功地将位置信息嵌入到通道注意力中，使网络能够对更广泛的区域进行注意力计算。然而，CA 的计算需求相对较高，在资源受限的物联网环境中存在挑战。



## 本文工作

大多数基于深度学习的物体检测器都是按照针对自然图像的物体检测所采用的网络架构来设计的。当物体具有正常的形状和比例时，它们能够取得良好的检测效果。然而，宽带频谱图中的信号往往具有极其拉长的形状，且在持续时间和带宽方面存在显著变化。此外，电磁环境变得越来越复杂[26]、[27]，并且充斥着各种各样的信号。这些极其密集和多样的信号也给快速准确的信号检测、估计和识别带来了挑战，这些挑战可以总结如下：

1. 信号的不确定锚定比例在物联网的背景下构成了一个关键挑战，因为在物联网中，精确的定位和准确的对象检测至关重要。在物联网环境中，信号具有动态特性，由于连接设备和通信协议的多样性，它们会在不同的频率和时间出现。与自然图像中的传统对象检测不同，物联网信号不受特定频段或可预测的时间模式的限制。这种锚定比例的不确定性在物联网复杂的电磁环境中表现得尤为明显，因为这些设备在具有复杂性和多变性的频谱中运行。随着信号在具有不可预测带宽的频谱中传输，准确检测变得尤为困难，因为难以建立固定的锚定比例。
2. 信号中存在极端的锚点比例这一情况构成了一个独特的挑战，尤其是在考虑到在物联网环境中广泛使用的基于深度学习的检测器时。许多现有的深度学习检测器传统上会指定特定的锚点比例，例如在自然图像中的物体上为 3：1 或 4：1。然而，在物联网领域，信号呈现出多样化的形状和持续不变的信号，特别是可能会与传统对象检测方法中使用的标准锚点比例有很大差异。如图 1 所示，信号的极端锚点比例非常明显，这导致了所指定的锚点比例与信号的真实特征之间可能存在不匹配的问题。这种差异在信号由于其独特的频率、时间或带宽特性而明显偏离预期比例的情况下尤为显著。



此外，图表底部左角的大量信号集中现象表明，在时频表示中同时存在众多小信号。经过分析，发现宽度小于 0.1 且高度小于 0.05 的信号占所有信号的 27.2%。这种弱信号的普遍存在为物联网环境中的频谱感知增加了另一层复杂性，因为信号的多样性和动态性要求采用创新方法以确保有效检测和定位。
