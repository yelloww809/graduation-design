## 2.2 数据集难点
1. 每条数据的点数都非常多（1e6级别），考验数据预处理；
2. 同一条数据中包含的信号数量可能较多（最多8个），且带宽相差很大（最小0.006MHz，最大40.0MHz），信号强度差异也很大，考验数据预处理；
3. 同一条数据内的信号存在时域和频域上的重叠（即干扰），考验检测模型；
4. 经过观察，Class 9 同一类型的不同带宽下的时频图特征不同，即存在不同特征不同模样的信号同属一种类型，考验数据预处理和检测模型；
